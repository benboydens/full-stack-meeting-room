# Full Stack Meeting Room

Gemaakt in opdracht van CityMesh.

## Dependencies

### Backend
- express
- mongodb atlas

### Frontend
- vue 2
- vuetify

## Install


### Frontend

Instal backend locally
```
npm install
```

Run development server
```
npm run dev
```

### Frontend

Instal frontend locally
```
npm install
```

Run development server
```
npm run serve
```

Build frontend
```
npm build
```
